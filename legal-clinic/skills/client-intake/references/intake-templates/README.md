# 执业领域接案模板

由冷启动时从教授的接案表格中填充。如未提供，`/client-intake` 使用 `client-intake/SKILL.md` 第 2 步中的默认问题集。

每个诊所处理的执业领域各一份文件：

- `immigration.md` — 身份、入境、历史申请、原籍国状况、家庭情况、刑事记录、时间线
- `housing.md` — 住房类型、发生了什么、租约/付款、居住条件、时间线
- `family.md` — 关系、子女、安全状况、现有法院命令、时间线
- `consumer.md` — 债务类型、谁在联系、文件情况、已提起的诉讼、时间线

每份模板包含：应问的问题、提问顺序，以及敏感信息处理说明（例如，移民案件中的刑事记录——说明询问原因）。
